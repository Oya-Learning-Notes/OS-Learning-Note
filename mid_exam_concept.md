# 操作系统

**操作系统的作用**

- OS作为用户和硬件系统之间的接口
- OS作为计算机系统资源的管理者
- OS实现了对于计算机资源的抽象

**操作系统的发展**

- 单道批处理系统
    - 系统资源无法得到充分利用
- 多道批处理系统
    - 资源利用率高
    - 系统吞吐量大
- 

**操作系统的目标**

- 方便
- 有效
- 开放
- 可扩充

**操作系统的特性:**

- 并发
- 共享
- 异步
- 虚拟

**批处理系统**

允许提交多个任务，无交互性。
